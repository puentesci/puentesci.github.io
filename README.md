# Puente Scientific - Professional Export Company Website

A modern, professional website showcasing Puente Scientific as a legitimate export company specializing in international trade and logistics solutions. Features futuristic design, smooth animations, and comprehensive business functionality.

🌐 **Live Website**: [https://puentescientific.com](https://puentescientific.com)

## Project Overview

This is a static website built for GitHub Pages deployment, designed to establish Puente Scientific's professional online presence in the international export industry.

## Project Structure

```
puentesci.github.io/
├── index.html                   # Main landing page with hero, services, and CTA
├── contact.html                 # Contact page with form and business info
├── privacy-policy.html          # Privacy policy and data protection
├── terms-of-service.html        # Terms of service and legal information
├── CNAME                       # Custom domain configuration
├── assets/                     # Static assets
│   ├── css/                   # Modular stylesheets
│   │   ├── reset.css          # CSS reset and normalization
│   │   ├── variables.css      # CSS custom properties and themes
│   │   ├── animations.css     # Animation keyframes and transitions
│   │   ├── components.css     # Reusable component styles
│   │   ├── main.css           # Main layout and page styles
│   │   └── responsive.css     # Mobile and tablet responsive styles
│   ├── js/                    # JavaScript functionality
│   │   ├── main.js           # Core application logic
│   │   ├── animations.js     # Animation controllers
│   │   ├── contact.js        # Contact form handling
│   │   └── analytics.js      # Google Analytics integration
│   └── images/               # Optimized images and graphics
└── docs/                      # Documentation
    ├── DEPLOYMENT.md          # Deployment and hosting guide
    └── ANALYTICS.md           # Analytics setup and tracking guide
```

## Quick Start

### Local Development
```bash
# Clone the repository
git clone https://github.com/puentesci/puentesci.github.io.git

# Navigate to project directory
cd puentesci.github.io

# Open with a local server (Python example)
python -m http.server 8000

# Visit http://localhost:8000 in your browser
```

### Deployment
The site is automatically deployed via GitHub Pages when changes are pushed to the `main` branch. See [docs/DEPLOYMENT.md](docs/DEPLOYMENT.md) for detailed deployment instructions.

## Content Management

### Adding New Content

2. **Styles**: Modify existing CSS files or add new ones in `assets/css/`
3. **Scripts**: Add JavaScript functionality in `assets/js/`
4. **Pages**: Create new HTML files in the root directory
